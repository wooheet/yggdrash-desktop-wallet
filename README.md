# yggdrash Wallet
Yggdrash Wallet Desktop App for YEED coin

## Download
[Latest Release]()


#### Homebrew
For Mac users the package is available in [Homebrew](https://brew.sh/):

```
brew update
brew install yarn

yarn add concurrently electron --dev
```

## Features
* Available on ***Windows***, ***Linux*** (Ubuntu/Debian) and ***MacOSX*** (signed).
